# Tour Virtual VR de Machu Picchu

## Descripción

Proyecto de Realidad Virtual desarrollado en **Unity 6** utilizando la herramienta **IndretsVR Virtual Tour**. La aplicación permite recorrer virtualmente distintos sectores de Machu Picchu mediante imágenes panorámicas de 360°, navegación por hotspots y paneles informativos con contenido histórico.

## Objetivo

Brindar una experiencia inmersiva que facilite el conocimiento y la exploración del patrimonio cultural de Machu Picchu a través de un recorrido virtual intuitivo e interactivo.

## Características

- Recorrido virtual mediante imágenes panorámicas 360°.
- Navegación entre escenas utilizando hotspots.
- Paneles informativos con datos históricos y culturales.
- Menú principal para iniciar el recorrido.
- Interfaz sencilla y de fácil uso.

## Tecnologías utilizadas

- Unity 6
- IndretsVR Virtual Tour
- C#

## Requisitos

- Windows 10/11 (64 bits)
- Procesador Intel Core i5 o equivalente
- 8 GB de memoria RAM (mínimo)
- Tarjeta gráfica compatible con DirectX 11
- Resolución recomendada: 1920 × 1080

## Ejecución

1. Abrir el proyecto con Unity 6.
2. Ejecutar la escena **punto7**.
3. Elegir cualquiera círculo de los hotspot.
4. Navegar entre los diferentes panoramas utilizando los hotspots.
5. Consultar la información histórica mediante los paneles informativos.

## Estado del proyecto

Versión final desarrollada como proyecto académico. Todas las funcionalidades planificadas fueron implementadas y validadas mediante pruebas funcionales.
